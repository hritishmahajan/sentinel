// What it does

Everything a production AI system needs

Every request through Sentinel goes through a 7-step governed pipeline before it reaches an LLM — and produces a complete audit trail on the way back.

🔀
Multi-provider routing
Unified API across Anthropic, OpenAI, and xAI Grok. Automatic failover when a provider's circuit opens. Redis-backed circuit breaker shared across all gateway instances.
🏢
Per-tenant governance
Model allowlists, denied topics, token ceilings, cost budgets, and request-per-minute limits — all configurable per tenant with zero code changes.
🛡️
Prompt injection defense
Curated pattern set with noisy-OR risk scoring. Tuned against a labelled golden dataset. Precision and recall gated in CI — can't regress silently.
🔒
PII redaction
Emails, phone numbers, SSNs, AWS keys, Luhn-validated credit cards — all redacted before leaving your network. Typed placeholders in the audit log, never the original values.
📋
Append-only audit log
Every request produces exactly one immutable row in Postgres — auth failures, policy denials, provider errors, successful completions. Keyset-paginated for the dashboard.
📊
Full observability
9 Prometheus metrics covering requests, latency, tokens, USD spend, circuit state, policy decisions, rate-limit hits, injection blocks, PII redactions. OTel-ready.
// How it works

Request lifecycle in 7 steps

Order is deliberate — cheap checks fail first. Every branch writes an audit row before raising an exception.

client ──→ [1. auth] ──→ [2. rate limit] ──→ [3. policy] ──→ [4. security] ──→ [5. router] ──→ LLM provider │ │ │ │ │ postgres redis in-memory in-memory redis (circuit) (api key hash) (token bucket) (policy eval) (regex + scan) + tenacity retry + failover [6. response] ←── provider ←── [5. router] │ [7. audit write] ──→ postgres (append-only, immutable) │ prometheus metrics + structlog JSON + otel traces

// Drop-in SDK

One import, full governance

The Python SDK is a drop-in replacement for the Anthropic SDK. Existing code moves behind the gateway with one line change.

Python SDK
curl
# pip install sentinel-sdk from sentinel_sdk import Sentinel client = Sentinel( api_key="sk_live_...", base_url="https://sentinel-gateway.railway.app", ) response = client.messages.create( model="grok-3-mini", max_tokens=512, messages=[{"role": "user", "content": "Hello"}], ) print(response.content) # "Hello! How can I help?" print(response.cost_usd) # 0.000127 print(response.provider) # "grok" print(response.request_id) # "a3f8b2..." → audit log
